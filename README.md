# MVSACE
  This folder includes simulation codes for the manuscript titled "Bayesian inference for survivor average causal effects in cluster-randomized trials with multivariate outcomes subject to complex missing data" Authors: Guangyu Tong*, Chenxi Li, Eric Velasquez,Shashi Harini, Newman P. Stanton, Tassos Kyriakides, Michael O. Harhay, Fan Li Files: This folder includes two files as follows:
  1. bivariate_continous_simulation_github.R： Data generating process，simulation code and result summary codes for bivariate continuous outcomes
  2. bivariate_binary_simulation_github.R： Data generating process，simulation code and result summary codes for bivariate binary outcomes
